This is a simple spaced repetition app designed to assist with organizing study or workflow tasks.
To add a task, simply click the button and a text box will appear.
If you'd like to cancel the current task you're adding, hit escape and the text box will be removed.
Otherwise, hit enter when you're happy with the task you've written.
